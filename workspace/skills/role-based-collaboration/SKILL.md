# 多角色协作协议

## 角色定义

| 角色 | 职责 | 触发场景 |
|------|------|----------|
| **笔杆子** | 写作、编辑、总结 | 需要写文章、改稿、摘要 |
| **参谋官** | 分析、建议、决策支持 | 需要建议、方案对比 |
| **运营官** | 执行、整理、维护 | 日常任务、文件整理 |
| **进化官** | 学习、研究、技能获取 | 遇到新问题、需要学新东西 |
| **健康管家** | 提醒、记录、跟踪 | 定时任务、健康相关 |
| **交易官** | 数据分析、监控、报告 | 股票、数据监控 |

## 执行原则

1. **识别任务类型** → 分配合适角色
2. **能并行则并行** → 同时启动多个子代理
3. **复杂任务分解** → 大任务拆小任务
4. **结果汇总汇报** → 整合各角色输出

## 启动命令

```bash
# 启动笔杆子
sessions_spawn(task="写作任务", label="笔杆子", mode="run")

# 启动参谋官
sessions_spawn(task="分析任务", label="参谋官", mode="run")

# 启动进化官研究新技能
sessions_spawn(task="研究并安装技能", label="进化官", mode="run")
```
