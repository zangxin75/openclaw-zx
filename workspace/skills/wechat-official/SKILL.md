# 微信公众号运营专家

一站式公众号内容生产、排版、发布、运营全流程支持。

## 工作流程

```
选题策划 → 内容创作 → 排版设计 → 发布检查 → 数据分析
```

## 核心能力

### 1. 选题策划 (`topic_research`)

**热点追踪**
- 监控行业热点、社会话题
- 分析竞品账号选题方向
- 结合时令/节日/事件策划专题

**选题评估标准**
| 维度 | 权重 | 评估问题 |
|------|------|----------|
| 受众匹配 | 30% | 我的读者会感兴趣吗？|
| 时效性 | 25% | 现在发是否有热度？|
| 差异化 | 20% | 能否提供独特视角？|
| 可操作性 | 15% | 我能写好吗？|
| 传播潜力 | 10% | 会被转发吗？|

**选题库维护**
- 长期选题：品牌故事、产品深度、行业洞察
- 短期选题：热点回应、节日营销、活动预告
- 储备选题：至少保持 5-10 个备用选题

### 2. 内容创作 (`content_creation`)

**文章结构模板**

**模板A：干货型**
```
标题：数字+痛点+解决方案
开头：痛点场景 → 共鸣建立
正文：3-5个要点，每点配案例
结尾：总结 + 行动号召 + 福利
```

**模板B：故事型**
```
标题：悬念/冲突/情感共鸣
开头：场景描写，引人入胜
正文：起承转合，细节丰富
结尾：点题 + 情感升华
```

**模板C：观点型**
```
标题：颠覆认知/金句/立场鲜明
开头：提出反常识观点
正文：论据1 → 论据2 → 反驳反面
结尾：强化立场 + 互动提问
```

**写作检查清单**
- [ ] 标题是否吸引人？（用数字、悬念、痛点）
- [ ] 开头前3秒能否抓住注意力？
- [ ] 段落是否过长？（手机阅读建议3-4行一段）
- [ ] 是否有金句/重点加粗？
- [ ] 结尾是否有明确的CTA？
- [ ] 全文是否有3个以上互动点？

### 3. 排版设计 (`formatting`)

**微信生态规范**
- 字号：标题16-18px，正文14-16px，注释12px
- 行距：1.5-1.75倍
- 颜色：主色不超过3种，重点用品牌色
- 边距：左右留白10-15px

**排版结构**
```
封面图（900×383px 或 500×500px）
↓
引导关注（头图/动图）
↓
正文（段落+配图+分割线）
↓
结尾引导（点赞/在看/关注）
↓
作者介绍
↓
历史文章推荐
↓
二维码/公众号名片
```

**排版工具**
- 秀米 XIUMI
- 135编辑器
- 壹伴助手
- Markdown Here

### 4. 发布检查 (`publish_check`)

**发布前必检**
- [ ] 标题：无错别字，无敏感词
- [ ] 封面图：清晰度，版权，视觉吸引力
- [ ] 摘要：是否自动提取？需不需要手动优化？
- [ ] 阅读原文链接：是否需要？是否正确？
- [ ] 留言功能：是否开启？
- [ ] 群发对象：全部用户/标签分组？
- [ ] 定时发送：是否设置？时间是否合适？

**最佳发布时间**
| 类型 | 推荐时间 | 说明 |
|------|----------|------|
| 早报 | 07:00-09:00 | 通勤时间 |
| 干货 | 12:00-13:00 | 午休时间 |
| 深度 | 20:00-22:00 | 睡前阅读 |
| 热点 | 即时或18:00-19:00 | 下班后 |

### 5. 数据分析 (`analytics`)

**关键指标**
- 打开率：阅读量/送达人数（健康>10%）
- 分享率：分享次数/阅读量（健康>5%）
- 完读率：读完人数/阅读人数（健康>20%）
- 增粉率：新增关注/阅读人数
- 取关率：取消关注/总关注

**分析维度**
- 哪类选题表现最好？
- 什么发布时间效果最好？
- 标题风格对打开率的影响？
- 文章长度对完读率的影响？

**迭代优化**
- 每月复盘：数据回顾 + 选题方向调整
- A/B测试：标题、封面、发送时间
- 用户调研：评论区互动、问卷、私聊

## 使用示例

### 场景1：帮我策划下周选题
```
用户：帮我策划下周的3个选题，行业是AI/科技
输出：
1. 热点选题：[具体热点] + 切入角度 + 预期效果
2. 干货选题：[具体痛点] + 解决方案 + 资料来源
3. 故事选题：[人物/案例] + 叙事角度 + 情感共鸣点
```

### 场景2：帮我写一篇文章
```
用户：写一篇关于[话题]的文章，面向[受众]，风格[专业/轻松/故事]
流程：
1. 确认选题和角度
2. 撰写大纲
3. 分段撰写正文
4. 优化标题和开头
5. 输出完整文章
```

### 场景3：排版优化
```
用户：帮我把这篇文章排版成适合公众号的格式
操作：
1. 调整段落长度
2. 添加小标题和重点加粗
3. 建议配图位置
4. 输出微信排版格式
```

| 风格 | 特点 | 适用账号 |

## 注意事项

1. **版权合规**：图片、字体、引用内容需确认版权
2. **敏感词规避**：政治、色情、谣言、竞品攻击
3. **原创声明**：重要文章申请原创保护
4. **互动维护**：发布后2小时内积极回复评论
5. **矩阵运营**：不同平台内容适配（公众号/视频号/知乎/小红书）
