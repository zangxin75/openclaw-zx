# Smart Task Router (STR) - 智能任务路由系统

## 功能概述

基于 oh-my-claudecode 架构启示，为 OpenClaw 实现智能任务路由系统：

1. **关键词触发** - 检测 `并行:` `团队:` `深度:` `自动:` 前缀自动选择执行策略
2. **任务复杂度评估** - 自动判断任务难度选择模型层级
3. **多模式执行** - 直接执行 / 并行代理 / 深度规划+验证
4. **验证循环** - 结果质量检查，不合格自动修复

## 使用方式

```
并行: 分析这3个文件的内容差异
团队: 帮我设计一个用户认证系统
深度: 重构这个项目的API架构
自动: 修复所有TypeScript错误
```

## 执行模式

| 关键词 | 模式 | 特点 |
|--------|------|------|
| `自动:` | Auto Mode | 自动判断复杂度，简单直接执行，复杂走完整流程 |
| `并行:` | Parallel Mode | 启动多个子代理并行处理不同方面 |
| `团队:` | Team Mode | 规划→执行→验证→修复 完整管道 |
| `深度:` | Deep Mode | 先深度分析规划，再执行，最后验证 |

## 模型分层

| 复杂度 | 模型 | 适用场景 |
|--------|------|----------|
| 低 | 轻量模型 | 简单查询、文件查找、格式转换 |
| 中 | 标准模型 | 常规编码、文档撰写、测试 |
| 高 | 强力模型 | 架构设计、复杂重构、性能优化 |
