# 内容运营专家配置

## 专用模型配置

### 公众号长文写作
```json
{
  "agentId": "content-writer",
  "model": "anthropic/claude-3-5-sonnet-20241022",
  "thinking": "verbose"
}
```
**优势**：中文写作最流畅，结构清晰，有深度洞察

### 短视频脚本
```json
{
  "agentId": "video-script-writer", 
  "model": "openai/gpt-4o",
  "thinking": "on"
}
```
**优势**：节奏感强，懂短视频爆点，金句多

### 小红书运营
```json
{
  "agentId": "xiaohongshu-writer",
  "model": "anthropic/claude-3-5-sonnet-20241022"
}
```
**优势**：网感好，emoji自然，懂小红书调性

## 多平台内容策略

### 内容金字塔
```
            公众号（深度原创）
                   ↓
    ┌──────────────┼──────────────┐
    ↓              ↓              ↓
 视频号         抖音            小红书
（5分钟）    （1分钟）      （图文+短文案）
   解读          切片           干货图
```

### 一稿多改流程

**源头**：公众号深度长文（2000-3000字）

**改视频号**：
- 提取3个核心观点
- 每个观点扩成1-2分钟口述
- 配案例/数据/画面

**改抖音**：
- 找1个最爆的点
- 前3秒抓眼球
- 中间快节奏输出
- 结尾引导关注

**改小红书**：
- 提炼干货清单（3-5条）
- 配高颜值封面图
- emoji分段
- 标签优化

## 选题策划流程

### 周一：选题会
1. 扫描AI行业最新动态（产品发布、融资、政策）
2. 分析上周数据，找出高表现内容方向
3. 确定本周3个核心选题

### 周二-周四：内容生产
- 周二：公众号深度文
- 周三：视频号脚本
- 周四：小红书图文

### 周五：发布排期
- 优化标题/封面
- 设置定时发布
- 准备互动回复话术

## 数据复盘指标

### 公众号
- 打开率（目标>15%）
- 分享率（目标>8%）
- 完读率（目标>25%）

### 视频号
- 完播率（目标>30%）
- 互动率（点赞+评论/播放）

### 抖音
- 3秒完播率（目标>50%）
- 整体完播率（目标>15%）

### 小红书
- 互动率（点赞+收藏+评论/曝光）
- 涨粉率

## 使用方式

需要创作内容时，指定模型：

```
"/model anthropic/claude-3-5-sonnet" 
然后：帮我写一篇关于[话题]的公众号文章
```

或启动专门代理：
```
"启动内容运营专家，用claude-3-5-sonnet模型"
```
