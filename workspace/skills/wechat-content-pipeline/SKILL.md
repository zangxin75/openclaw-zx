## 三特工协作模式

### 特工1：热点雷达（热点追踪员）

**职责**: 监控全网AI相关热点

**监控渠道**:
- Twitter/X: @sama, @ylecun, @DrJimFan, @karpathy
- 即刻: AI主题圈子
- 知乎: AI相关热榜
- 公众号: 机器之心、量子位、AI前线
- GitHub: Trending repositories
- Product Hunt: 每日新品
- 36kr/虎嗅: 融资新闻

**输出格式**:
```
热点名称: [一句话概括]
热度指数: ⭐⭐⭐⭐⭐
信息源: [链接/账号]
与定位关联度: 高/中/低
可切入角度: [如何结合实体赋能]
建议选题: [具体标题方向]
```

---

### 特工2：选题策划（选题架构师）

**职责**: 将热点转化为可执行选题

**筛选标准**:
| 维度 | 权重 | 标准 |
|------|------|------|
| 受众匹配 | 30% | 中小企业主/个体户能看懂、用得上 |
| 实操性 | 25% | 能给出具体操作步骤 |
| 时效性 | 20% | 当前热点或常青话题 |
| 差异化 | 15% | 不是简单转载，有独到视角 |
| 传播性 | 10% | 有转发价值 |

**选题模板**:
```
选题名称: [一句话]
核心观点: [我们要表达什么]
目标受众: [具体人群画像]
痛点场景: [他们遇到什么具体问题]
解决方案: [AI如何解决这个问题]
实操路径: [1、2、3步骤]
预期效果: [用数据说话]
案例支撑: [真实或虚构但合理的案例]
```

---

### 特工3：内容匠人（撰稿人）

**职责**: 将选题写成高质量文章

**写作流程**:
1. 大纲确认（结构清晰）
2. 标题优化（3个备选）
3. 正文撰写（分段输出）
4. 金句提炼（3-5个）
5. 案例补充（真实具体）
6. 结尾CTA（明确行动号召）

**文章结构模板**:
```
【标题】数字+痛点+承诺（如：3个AI工具，让我客服成本降70%）

【开头】场景描写（100字内抓住注意力）
  → 痛点共鸣：你是不是也遇到...
  → 结果预告：这篇文章给你...

【正文】3-5个核心要点
  每个要点：观点 → 解释 → 案例 → 实操步骤

【案例】具体场景+数据+前后对比

【实操】123步骤，每步配工具/截图说明

【避坑】常见错误 + 解决方案

【结尾】总结+福利+互动
  → 核心观点回顾
  → 免费资源赠送（引导关注）
  → 评论区提问互动
```

**质量检查清单**:
- [ ] 开头前3秒能否抓住注意力？
- [ ] 每段是否控制在3-4行（手机阅读）？
- [ ] 是否有具体数字/案例/数据？
- [ ] 实操步骤是否可立即执行？
- [ ] 是否有3个以上金句？
- [ ] 结尾是否有明确CTA？
